## Courses & Projects

### Python Programming

This Python course teaches you project-based, practical, hands-on python, with AI. I will show you how I would learn python in the "age of AI" to teach you how I would build all my old projects, nowadays, if I were starting over.

### Databases 101 & Legacy Data Integration with AI

This course takes students through your original process of building middleware with Excel and Access to integrate disparate data, then contrasts it with modern, AI-enhanced solutions.

### Cybersecurity AI for Healthcare and Finance

This course dives into high-value, regulated industries, focusing on using data science and AI to solve security challenges.

- **HealthTech Focus**: Protecting sensitive patient data (PHI - Protected Health Information) and securing medical devices.
- **FinTech Focus**: Securing transactional data, preventing sophisticated cyber-attacks, and meeting regulatory compliance.

### Learning Data Science in the Age of AI

This final course is an essential capstone, preparing students for the modern reality of the data science field where AI tools are commonplace. 

**Focus**: Shifting from building everything from scratch to using Generative AI (GenAI) to accelerate the entire data science lifecycle.
